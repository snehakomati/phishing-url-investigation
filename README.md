# Phishing URL Investigation

## Project Overview

This project demonstrates a phishing URL investigation workflow using threat intelligence and phishing analysis tools. The investigation focused on identifying malicious infrastructure, suspicious redirects, phishing indicators, and associated network activity.

The project involved collecting a phishing sample from PhishTank, validating the threat using VirusTotal, analyzing infrastructure behavior with URLscan.io, and extracting Indicators of Compromise (IOCs) for security investigation purposes.

---

## Tools Used

* PhishTank
* VirusTotal
* URLscan.io

---

## Investigation Objectives

* Identify phishing-related infrastructure
* Analyze malicious URL behavior
* Investigate redirects and network activity
* Extract Indicators of Compromise (IOCs)
* Document findings in a professional SOC-style report

---

## Key Findings

* The investigated domain exhibited phishing-related naming patterns.
* VirusTotal identified multiple malicious detections associated with the URL.
* URLscan.io analysis revealed redirect behavior to a suspended-domain page.
* Multiple associated IP addresses and infrastructure indicators were identified during the investigation.
